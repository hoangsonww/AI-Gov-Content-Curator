# Validation Map

- `backend/`: `cd backend && npm test`
- `frontend/`: `cd frontend && npm run lint`; add Playwright for route and interaction changes
- `crawler/`: `cd crawler && npm test`
- `newsletters/`: `cd newsletters && npm test`
- `agentic_ai/`: consistency review first; run commands only when the Python environment is ready
- `infrastructure/`: consistency review first; run apply/deploy commands only in a prepared environment
